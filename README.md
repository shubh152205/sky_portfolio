# 🌌 Sky Portfolio — Shubham Sharma

A stunning, immersive 3D portfolio website built with **Next.js**, **React Three Fiber**, and **GSAP**. Navigate through a cinematic sky-themed experience featuring floating clouds, stars, and interactive portals to explore my work and projects.

## ✨ Features

- **3D Immersive Experience** — Full WebGL-powered scene with dynamic camera movements
- **Interactive Portals** — Click-to-enter tiles for Work/Education and Side Projects
- **Animated Timeline** — A 3D scrollable work history with smooth curve-based navigation
- **Theme Switcher** — Dynamic colour themes that morph the entire scene
- **Responsive Design** — Optimised for both desktop and mobile devices
- **Noise Overlay** — Subtle grain texture for a cinematic aesthetic

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| [Next.js](https://nextjs.org/) | React framework with SSR & routing |
| [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) | Declarative 3D rendering |
| [Three.js](https://threejs.org/) | WebGL 3D library |
| [DREI](https://github.com/pmndrs/drei) | R3F helpers (Text, ScrollControls, etc.) |
| [GSAP](https://greensock.com/gsap/) | High-performance animations |
| [Zustand](https://zustand-demo.pmnd.rs/) | Lightweight state management |
| [Tailwind CSS](https://tailwindcss.com/) | Utility-first styling |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/shubh152205/sky_portfolio.git
cd sky_portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## 📦 Build for Production

```bash
npm run build
npm start
```

## 🌐 Deployment

This project can be deployed on [Vercel](https://vercel.com) with zero configuration — just connect your GitHub repo and it auto-deploys on every push.
